Research library

Research Proposal Tutor Library

This library helps students test whether a dissertation or research proposal is focused, researchable and feasible.

Download this librarySee examplesBack to tools

What students can get from it

This library gives structured feedback and questions. It does not aim to produce a finished assignment. The student should decide what to use and revise the work themselves.

Tools included

ToolUse it when...What you get
Research Question, Aim and Objectives CheckerUse it when the project has a topic but not a clear question.Checks alignment and focus.
Methodology Fit CheckerUse it when methods may not match the question.Advice on method, data, sample and feasibility.
Critical Research Supervisor ReviewUse it before polishing a proposal.A direct supervisor-style critique.
Viva or Supervisor PracticeUse it to practise explaining the project.One supervisor-style question at a time.
Guided Topic BrainstormingUse it early in topic development.Questions first, then possible project ideas.

Mini example

Student input

I want to study how social media affects young people and use interviews, discourse analysis and netnography.

Typical tutor response

The topic is too broad for a dissertation proposal. A supervisor would ask: which platform, which group of young people, which type of “effect”, and what data will you actually analyse? The methods also need narrowing.

What this shows

The tool acts like a critical supervisor. It checks feasibility before the student spends time polishing sentences.

This is a short illustration. Actual outputs are longer, more structured and in Markdown.

Precision before polish

These tools should not make work sound more academic by changing the student’s concepts. If a suggested term may change the meaning, the tutor should explain the difference and ask the student to choose.
