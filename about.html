About this site

The AI Personal Tutor Toolkit is a public collection of Markdown prompt libraries designed to help students use AI as a tutor, not as a ghost-writer.

These prompts were created by Dr Mark Beachill.

The GitHub repository is available at github.com/markbeachill/tutorprompts.

Questions, suggestions and discussion can be posted in the repository's GitHub Discussions area.

What the toolkit is for

The toolkit is designed for learning-focused AI support: feedback, explanation, practice, revision planning and clearer student thinking.

It is not designed to guarantee academic integrity or prevent misuse. Its purpose is to make responsible, student-owned AI use easier.
