# AI Personal Tutor Toolkit — site update v2.10

Base package: `ai_personal_tutor_github_pages_site_v2_9_st1_update.zip`

## Summary

This maintenance update applies the WT5 audit findings to the full site package. It preserves the ST1 v2.9 update and adds WT5 boundary, vague-meaning, location and strong-passage teaching improvements.

## Version changes

- Site package: v2.10
- Prompt libraries: v2.3
- Testing/audit pack: v2.7

## Main WT5 changes

- Replaced default `Suggested improvement` pattern with `Move to make` and `Your turn`.
- Added required location information for each WT5 improvement.
- Added a vague-meaning rule so WT5 does not specify the student's research focus on their behalf.
- Added guidance against multiple submission-ready replacement sentences in one review.
- Added a strong-passage teaching rule so students identify and transfer successful writing moves from their own work.
- Added follow-up guidance to avoid repeated diagnostic dependency.
