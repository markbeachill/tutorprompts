Writing library

Writing Tutor Library

This library helps students improve wording, sentence control, paragraph clarity, mistakes, style and referencing without asking AI to write the assignment for them.

Download this librarySee examplesBack to tools

What students can get from it

This library gives structured feedback and questions. It does not aim to produce a finished assignment. The student should decide what to use and revise the work themselves.

Tools included

WT2 now works from paragraph logic before polish: it helps students connect ideas before rewriting a topic sentence.

Mini example

Student input

The implementation of influencer marketing has had an impact on the development of brand engagement among young consumers.

Typical tutor response

The sentence makes sense, but it is heavy. The main action is hidden inside nouns such as “implementation” and “development”. A clearer sentence would show who is doing what. Try making the actor and verb clearer: “Brands use influencer marketing to build engagement among young consumers.”

What this shows

The tool does not just polish the sentence. It explains the pattern: hidden action, heavy nouns and unclear actors.

This is a short illustration. Actual outputs are longer, more structured and in Markdown.

Precision before polish

These tools should not make work sound more academic by changing the student’s concepts. If a suggested term may change the meaning, the tutor should explain the difference and ask the student to choose.
