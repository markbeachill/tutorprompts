Changelog

Version history.

A short record of major changes to the site, prompt libraries and testing pack.

Site v2.7 — formatting and audience cleanup

  • Fixed page structure so panels remain inside the standard page container.
  • Removed developer-facing folder details from the homepage.
  • Added editing notes and a CSS guardrail to reduce layout breaks in future edits.
  • Kept version-folder details on maintainer-facing pages only.

Site v2.6 — version-folder structure

Download folders were reorganised so current files live in prompt-libraries/latest/ and audit-library/latest/, while fixed release copies live in prompt-libraries/v2.0/ and audit-library/v2.4/.

Site v2.5 — WT2 paragraph logic update

  • Updated WT2 Single Paragraph Analysis so it starts with the paragraph’s chain of ideas, missing links and “so what?” questions.
  • Added controlled modelling rules for paragraph-level teaching.
  • Updated the testing pack with a two-turn WT2 test and new WT2 audit criteria.

Site v2.4 — precision before polish

Added global rules and site guidance to prevent academic-sounding wording from changing the student’s intended meaning.

Site v2.3 — clinic-style teaching refinement

Clinic-style tools now use made-up before/after examples before asking the student to revise their own wording.

Site v2.2 — interaction types and testing update

Added a distinction between interactive tutoring tools and full review tools, and updated audit criteria accordingly.

Site Stage 4 v1.0

Expanded the testing and audit area with a workflow, testing-file descriptions, minimum testing recommendations, audit criteria and release checklist.

Site Stage 3 v1.0

Added example input/output sections so users can preview what each mini library does before downloading.

Library v2.0

Added visible version information near the top of each library and version history sections at the bottom.
