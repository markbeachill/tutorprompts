AI Personal Tutor Toolkit

Purpose: These prompt libraries help students use AI as a tutor: to get feedback, ask better questions, practise skills, and revise their own work. They are not designed to write assessed work for students.

Download all mini libraries Download master library

How students use it

Upload one Markdown library to an AI tool, type prompt, choose a menu item, then paste or upload the work you want help with.

Outputs are in clean Markdown by default, so they are light, copyable, and easy to save.

Choose a library

The mini libraries are recommended for students, especially on free AI plans. The master library contains everything in one file and is better for editing, staff use, or stronger plans.

Quick start

Download

Choose one mini library, or download the ZIP if you want all five.

Upload

Upload the Markdown file to your AI tool. You can also upload a working document if your plan allows it.

Type prompt

The tutor will show the menu for that library. Choose one tool at a time.

Paste your work

Use your own sentence, paragraph, section, draft, feedback, or proposal. Revise it yourself afterwards.

Download ZIP

Free-plan tip

For free AI plans, use a mini library and work in small chunks. Plain text or Markdown usually works better than large Word or PDF files. Paste one paragraph, one section, or up to about 800 words at a time. Outputs are in Markdown by default because this is lighter and easier to copy.

Responsible use

This toolkit is for students who want to learn. It gives feedback, questions, examples, teaching material and revision guidance. It should not be used to produce final submission-ready work.

Use it to learn

Ask for feedback, explanations, questions, practice tasks and revision priorities.

Stay responsible

You decide what to change. You are responsible for the work you submit.

Keep a record

Use the AI-Use Record tool if your course asks you to explain how you used AI.
