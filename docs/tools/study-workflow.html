Mini example

Student input

My tutor said my essay is too descriptive, my paragraph order is confusing, and I need to define key terms.

Typical tutor response

Start with the argument, not the wording. Priority 1: define the key terms. Priority 2: reorder the paragraphs so the argument builds clearly. Priority 3: add analysis after descriptive sections.

What this shows

The tool helps the student decide what to do first instead of trying to fix everything at once.
