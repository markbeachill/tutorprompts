Single-tool downloads

Use one tutor tool at a time.

Single-tool downloads are a smaller backup option for students using free AI accounts, tools with small upload limits, or chats that struggle with the mini libraries.

Each download is one Markdown file. Choose the tool that matches the problem you are working on, upload or paste that file into your AI chat, then add your own writing, feedback, brief or question.

Current downloads use Prompt libraries v4.0. These files are generated from the same tutor-tool source as the mini libraries.

When to use this page

Start with a mini library when your AI tool can handle it. Use single tools when the AI says the file is too large, forgets the menu, behaves inconsistently, or you only need one focused kind of help.

This route is especially useful on free plans. It reduces the amount of toolkit text in the chat, leaving more room for the student’s own work and the tutor conversation.

How to use a single-tool file

  1. Pick the tool that best matches the current problem.
  2. Download the Markdown file for that tool.
  3. Upload or paste it into your AI chat and ask the AI to use only that tool.
  4. Paste the assignment brief, paragraph, draft extract, feedback or research idea the tool asks for.
  5. Use the feedback to revise the work yourself. Say “I’m stuck” if you need the tutor to slow down.
