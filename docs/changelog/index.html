Site v2.10 / Libraries v2.3 / Testing pack v2.7

Site v2.9 / Libraries v2.2 / Testing pack v2.6

  • Added guidance on handling marker, tutor or supervisor feedback as evidence of reader confusion, not as a marker-response service.
  • Updated the audit prompt and ST1 test card so this central-claim issue is checked in future regression testing.
