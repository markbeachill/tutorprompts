# Site update v3.8 — AI setup and student setup guidance

## Release

- Site package: v3.8
- Prompt libraries: v3.4
- Testing/audit pack: v3.4

## Summary

This site-only release updates the setup guidance after review of the v3.7 package. It treats the full v3.7 site package as the target and does not alter prompt-library or testing-pack behaviour.

## Changes

- Replaced the setup placeholder page with **Tutor Guide: Choosing and Setting Up an AI**.
- Added advice that tutors should test planned classroom activities themselves using the same or closest available setup students will use.
- Added guidance on access, institutional systems, choosing or specifying thinking/reasoning models, free-plan limits, optional Projects/workspaces and shared custom AIs/public GPTs.
- Simplified the Student guide so it no longer repeats detailed toolkit instructions.
- Added concise student guidance on experimenting with available AI systems, using thinking/reasoning modes where available, avoiding free-plan usage problems and optionally using Projects or workspaces.
- Removed the stray setup-page text that told users what to type if the AI summarised the toolkit.

## Behavioural impact

No prompt-library behaviour changed. This is a public-site guidance update.
