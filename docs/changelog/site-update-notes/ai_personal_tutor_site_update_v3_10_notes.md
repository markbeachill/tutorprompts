# AI Personal Tutor Toolkit — site update v3.10 notes

## Release label

Site package: v3.10  
Prompt libraries: v3.4  
Testing/audit pack: v3.4

## Summary

This is a site-only documentation update. Prompt-library behaviour and the testing/audit pack are unchanged.

## Changes

- Reorganised `docs/guides/tutors.html` so the page moves from toolkit purpose and boundaries, to rules and privacy, to recommendation/classroom use, student agency, library choice and testing.
- Added a `How to recommend it to students` subsection with suggested tutor wording that presents the toolkit as structured feedback, revision and learning support without overriding course, university or assessment rules.
- Added a shorter classroom wording option for tutors introducing the toolkit.
- Added a `Tell students they can disagree` section explaining that students can correct the AI, ask it to reread, explain what they meant or check with a human tutor.
- Renamed the library-choice section to `What library to use` while keeping the smallest-useful-library principle.
- Simplified `docs/examples/index.html` to one Writing Tutor WT1 example based on an unedited real-world response to a student sentence.
- Moved the `Choosing and setting up an AI` card up one position on the Guides page.

## Unchanged

- Prompt libraries remain v3.4.
- Testing/audit pack remains v3.4.
