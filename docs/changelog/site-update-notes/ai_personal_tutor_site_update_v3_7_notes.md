# Site update v3.7 — Teaching approach and Student guide rewrite

## Versions

- Site package: v3.7
- Prompt libraries: v3.4
- Testing/audit pack: v3.4

## Summary

This site-only release rewrites two important reading pages to better match the v3 paragraph-first design direction.

- Rewrote `docs/guides/teaching-approach.html` around writing as thinking, ideas first, manageable feedback, “I’m stuck” support and development rather than completion.
- Kept the references to Orwell, Williams and plain English principles in the Teaching approach page.
- Rewrote `docs/student-help/students.html` with a more direct student-facing explanation of what the toolkit gives students and how to use it.
- Fixed Study Workflow code as `SW`.
- Kept pushback, “I’m stuck”, good-feedback, privacy and responsibility guidance visible.

No prompt-library or testing/audit behaviour changed in this release.
