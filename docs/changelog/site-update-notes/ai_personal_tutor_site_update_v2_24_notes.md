# Site update v2.24 — prompt-library maintenance

Site package: v2.24  
Prompt-library suite: v2.5

## Summary

This release updates the current prompt-library downloads while leaving the testing/audit pack at v2.9.

## Prompt-library changes

- Changed visible file status from “working draft” to “active public release”.
- Added global guidance to adapt feedback detail, vocabulary, examples and expectations to the student's stated level, discipline and task.
- Updated WT6 so the “use the toolkit house style” fallback is more prominent for students who do not know their local referencing guide.
- Added a WT3 worked correction-boundary example showing when to correct directly and when to explain the issue instead of supplying a full rewritten sentence.
- Updated AT10 wording from “random topic” to “useful starting point” so the Socratic Tutor selects a purposeful discussion focus from the student's work.
