# Site update v3.9 — Student Help structure fix

## Release

- Site package: v3.9
- Prompt libraries: v3.4
- Testing/audit pack: v3.4

## Summary

This site-only release corrects the Student Help structure after the v3.8 setup guidance update. It restores the main Student guide as the first student-facing help page and moves the student AI setup material into its own page.

## Changes

- Restored the main **Student guide** at `docs/student-help/students.html`.
- Moved the v3.8 student AI setup guidance to `docs/student-help/student-ai-setup.html`.
- Updated the Student Help home page so the Student guide card says: “What this is and how to get started.”
- Added a separate Student AI setup guide card for choosing and setting up an AI system before using the toolkit.

## Behavioural impact

No prompt-library behaviour changed. This is a public-site structure and guidance update.
