# AI Personal Tutor Toolkit — site update v3.11 notes

## Summary

This release adds new guidance on using the toolkit repeatedly across a piece of work, staying in charge when AI output is imperfect, and building transferable AI-literacy skills. It also fixes confirmed library and testing-pack hygiene issues.

## Site content changes

- Added two sections to `docs/student-help/students.html`: `Use it again and again across one piece of work` and `The AI will not be perfect — stay in charge`.
- Added `docs/student-help/directing-ai.html`, a new Student Help page on learning to direct AI well.
- Added a Student Help home card for the new AI-skills page.
- Added `The AI-literacy argument` to `docs/guides/why-educators.html`.
- Added tutor guidance on honest expectations about AI behaviour and recommending the toolkit across the life of a piece of work.
