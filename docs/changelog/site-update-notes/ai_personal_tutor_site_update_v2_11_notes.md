# AI Personal Tutor Toolkit — site update v2.11

Base package: `ai_personal_tutor_github_pages_site_v2_10_wt5_update.zip`

## Release summary

This is a maintenance update based on the WT3 audit report. The audit judged WT3 safe to keep, with minor refinements around correction boundaries, plain-English grammar explanations, and explicit acknowledgement when a student correctly challenges a flagged mistake.

## Version changes

- Site package: v2.10 → v2.11
- Prompt libraries: v2.3 → v2.4
- Testing/audit pack: v2.7 → v2.8

## Main WT3 changes

- Added a correction-boundary rule: simple errors may receive direct corrections, but complex clause-level or sentence-level fixes should usually be explained rather than supplied as near-complete replacement sentences.
- Added a plain-English grammar-note rule: grammar explanations should be understandable without specialist terminology.
- Added a student-challenge protocol: when a student correctly disputes a flagged mistake, WT3 should acknowledge this explicitly before revising the list.
- Added guidance for responding calmly and non-defensively to frustrated but legitimate student pushback.
