# AI Personal Tutor Toolkit — Site update v3.2 notes

## Summary

Stage 4 of the v3 revision updates the testing/audit pack so it can evaluate the new v3 prompt-library behaviours.

## Versions

- Site package: v3.2
- Prompt libraries: v3.0
- Testing/audit pack: v3.0

## Main changes

- Added v3 audit criteria for paragraph-first tutor style, manageable feedback, writing as thinking, plain-English grammar explanations and “I’m stuck” support.
- Added U5 “I’m stuck” universal test.
- Added v3 regression cards for WT1 paragraph-first grammar explanation and manageable-feedback behaviour.
- Updated the educator testing guide and testing page to explain the v3 testing focus.
- Rebuilt latest and fixed v3.0 testing-pack downloads.

## Behaviour changes

No prompt-library behaviour changed in this release. The testing/audit pack was updated to test the v3.0 prompt-library behaviours introduced earlier.
