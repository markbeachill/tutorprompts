# AI Personal Tutor Toolkit — Site update v3.3

## Summary

Startup activation and launcher maintenance release.

## Versions

- Site package: v3.3
- Prompt libraries: v3.1
- Testing/audit pack: v3.1

## Changes

- Added explicit activation instructions at the top of every prompt library.
- Simplified mini-library and master-library launchers.
- Removed horizontal divider clutter from student-facing launchers.
- Added selected-tool start prompt guidance: ask only for the minimum input needed.
- Updated tool start prompts to use “paste or upload” where appropriate.
- Updated WT1 output style to use paragraph-first teaching responses, not numbered report sections or bullet lists.
- Deferred WT1 “More options” until the student asks for more options or says they are stuck.
- Added U6 Startup activation test to the testing pack.
- Updated audit prompt, educator testing guide and testing page for startup activation.

## No change

The core tool set remains unchanged. This release improves activation, startup and output style behaviour.
