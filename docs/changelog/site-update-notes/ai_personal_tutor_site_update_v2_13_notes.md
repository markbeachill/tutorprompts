# AI Personal Tutor Toolkit — v2.13 version-clarity maintenance release

Date: 2026-06-05

## Source base

Built from `ai_personal_tutor_github_pages_site_v2_12_testing_header_fix.zip`.

## Purpose

This release responds to the product-level review finding that the public site used suite-level labels such as Libraries v2.4 and Testing pack v2.8, while some current downloadable files still displayed older internal headings. That made the distinction between suite versions and file versions unclear.

## Changes made

- Added visible release stamps to the current prompt-library Markdown files.
- Updated the current prompt-library visible headers and front matter to show v2.4.
- Added visible release stamps to the current testing/audit Markdown files.
- Updated the current testing/audit visible headings to show v2.8.
- Rebuilt the latest prompt-library ZIP and v2.4 archive ZIP with the stamped files.
- Rebuilt the latest testing-pack ZIP and v2.8 archive ZIP with the stamped files.
- Added `release_manifest.md`.
- Updated `README.md`, `testing.html` and `changelog.html` with version-clarity guidance.

## Behavioural changes

None. This is a packaging, documentation and release-signalling update only.
