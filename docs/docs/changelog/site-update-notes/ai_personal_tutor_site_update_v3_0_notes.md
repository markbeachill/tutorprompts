# Site update v3.0 — prompt-library tutor-style revision

## Main changes

This release applies Stage 2 of the v3 revision: the prompt-library update. It does not revise the full site copy or testing/audit standards beyond release records.

Prompt-library changes:

- Added paragraph-first output guidance.
- Added manageable-feedback guidance.
- Added the “I'm stuck” support model.
- Added “writing is thinking” as a guiding principle.
- Clarified the toolkit scope as writing, revision, academic-thinking, research-planning and study-workflow support, not a general-purpose homework-answer system.
- Added plain-English grammar-term guidance, including subject, verb and object examples.
- Updated version stamps and fixed archive paths to v3.0.
