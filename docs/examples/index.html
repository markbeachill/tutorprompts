Writing Tutor Library

Writing Tutor Library example

Student input

The implementation of influencer marketing has had an impact on the development of brand engagement among young consumers.

Sample output extract

The sentence makes sense, but it is heavy. The main action is hidden inside nouns such as “implementation” and “development”. A clearer sentence would show who is doing what. Try making the actor and verb clearer: “Brands use influencer marketing to build engagement among young consumers.”

What the student gets out of it

The tool does not just polish the sentence. It explains the pattern: hidden action, heavy nouns and unclear actors.

Structure Tutor Library

Structure Tutor Library example

Student input

Paragraph 1 gives background on social media. Paragraph 2 jumps to BTS and McDonald’s. Paragraph 3 explains netnography. Paragraph 4 gives the aim.

Sample output extract

The structure has useful parts, but the order may confuse the reader. The method appears before the research aim is clearly established. A clearer order would be: topic and problem, research aim/question, key cases, then methodology.

What the student gets out of it

The tool comments on sequence and reader orientation rather than rewriting the draft.

Academic Thinking Tutor Library

Academic Thinking Tutor Library example

Student input

Social media advertising clearly controls what young consumers buy.

Sample output extract

A critical reader may challenge “clearly controls”. The claim assumes a direct cause-and-effect relationship. You may need evidence and a more careful verb such as “influences”, “shapes”, or “contributes to”.

What the student gets out of it

The tool pressures the argument. It asks what is assumed, what evidence is needed and how the claim could be made more careful.

Research Proposal Tutor Library

Research Proposal Tutor Library example

Student input

I want to study how social media affects young people and use interviews, discourse analysis and netnography.

Sample output extract

The topic is too broad for a dissertation proposal. A supervisor would ask: which platform, which group of young people, which type of “effect”, and what data will you actually analyse? The methods also need narrowing.

What the student gets out of it

The tool acts like a critical supervisor. It checks feasibility before the student spends time polishing sentences.

Study Workflow Tutor Library

Study Workflow Tutor Library example

Student input

My tutor said my essay is too descriptive, my paragraph order is confusing, and I need to define key terms.

Sample output extract

Start with the argument, not the wording. Priority 1: define the key terms. Priority 2: reorder the paragraphs so the argument builds clearly. Priority 3: add analysis after descriptive sections.

What the student gets out of it

The tool helps the student decide what to do first instead of trying to fix everything at once.
